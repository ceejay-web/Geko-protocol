# Geko Protocols - Institutional Digital Asset Terminal

Geko Protocols is a world-class institutional-grade trading terminal designed for high-frequency digital asset management, real-time liquidity aggregation, and AI-driven market analysis.

## Features

- **Institutional Charting**: High-fidelity real-time candle data powered by Lightweight Charts.
- **Neural Advisor**: Integrated Gemini AI for deep market sentiment analysis and strategy generation.
- **Rigged Settlement Engine**: Advanced trade resolution logic with administrative overrides.
- **Multi-Chain Connectivity**: Support for EVM and Substrate-based wallet identities.
- **Liquid Aggregator**: Atomic swap interface with live provider rate comparison.

## Getting Started

1. Connect your institutional identity (MetaMask, Phantom, or Manual Node).
2. Monitor real-time asset telemetry in the main HUD.
3. Deploy AI-driven long/short strategies using the Neural Analysis panel.

## Technology Stack

- **Frontend**: React, TypeScript, Tailwind CSS
- **Charts**: Lightweight-Charts
- **AI**: Google Gemini API (Gemini 3 Pro)
- **State Management**: LocalStorage with BroadcastChannel sync
